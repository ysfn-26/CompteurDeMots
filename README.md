 Word Counter - Java Project

Objectif
Ce projet compte le nombre total de mots dans un fichier texte en Java.

Fonctionnement
- Lecture d’un fichier texte ligne par ligne
- Découpage de chaque ligne en mots
- Comptage du nombre de mots par ligne
- Addition du total global

 Concepts utilisés
- Lecture de fichiers (Scanner / File)
- Boucles (while)
- Tableaux (String[])
- Manipulation de chaînes de caractères
- Gestion d’erreurs (try/catch)

Fichier utilisé
- test.txt (contient le texte à analyser)

Exécution
Ouvrir le projet dans IntelliJ IDEA puis lancer `Main.java`.

Auteur
Projet réalisé dans un cadre d’apprentissage Java.
